# Smart Travel Recommender

A full-stack web application built with **React (Vite)**, **FastAPI**, and the **Google Maps Directions API**.  
It helps users plan routes, visualize them on an interactive map, and track their CO₂ savings from choosing greener travel modes.

---

## Features

- Interactive Map: Displays routes between any two points using OpenStreetMap tiles.  
- Live Geolocation: Automatically centers on the user’s current location (with permission).  
- Multi-Mode Travel: Choose between driving, walking, biking, or public transit.  
- CO₂ Savings Tracker: Calculates and displays how much carbon emission you save compared to driving.  
- Secure API Keys: Environment variables are used for safe Google API key handling.  

---

## Tech Stack

| Layer | Technology |
|-------|-------------|
| Frontend | React + Vite + React-Leaflet + Axios |
| Backend | FastAPI + Python-Dotenv + Requests |
| APIs | Google Maps Directions API, Geocoding API |
| Styling | Inline CSS (customizable) |
| Deployment | Works locally or on cloud (Render, Railway, etc.) |
